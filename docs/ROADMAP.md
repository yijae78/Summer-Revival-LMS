## 5. 성공 측정 기준

### 기술적 성공
- [ ] Lighthouse Performance: 90+
- [ ] Lighthouse Accessibility: 90+
- [ ] WCAG 2.2 AA: axe-core 에러 0
- [ ] 빌드 에러 0
- [ ] 핵심 기능 100% 동작
- [ ] 오프라인 출석 체크 → 온라인 동기화 성공

### 사용자 성공
- [ ] 참가 신청 → 조 배정 → 출석 체크 플로우 3분 이내 완료
- [ ] 관리자가 30분 이내에 행사 기본 설정 완료 가능 (BYOS + Zero-Data 대시보드)
- [ ] 교사가 30초 이내에 첫 출석 체크 수행 가능 (코치마크 투어)
- [ ] 모바일에서 모든 핵심 기능 사용 가능 (삼성 인터넷 포함)
- [ ] 60대 봉사자가 독립적으로 앱 사용 가능 (48px 터치, 16px 본문)
